# Risk Matrix

| ID | Riesgo | Impacto | Probabilidad | Nivel | Justificación |
|----|--------|---------|--------------|-------|---------------|
| R1 | El usuario no puede completar el registro por validaciones incorrectas | Alto | Alta | Crítico | Bloquea el acceso de nuevos usuarios |
| R2 | El sistema permite usuarios duplicados | Alto | Media | Alto | Genera inconsistencias en datos |
| R3 | Contraseña insegura o débil | Alto | Media | Alto | Riesgo de seguridad |
| R4 | No se envía correo de confirmación | Medio | Alta | Alto | Usuario no activa cuenta |
| R5 | Mensajes de error poco claros | Medio | Media | Medio | Mala experiencia de usuario |
| R6 | Se permiten datos inválidos | Alto | Alta | Crítico | Afecta calidad de datos |
