# Charters

## Charter 1
**Título:** Exploración del flujo de compra y carrito

**Misión:** 
Explorar el flujo completo de compra y comportamiento del carrito, utilizando diferentes combinaciones de productos y acciones (agregar, eliminar, repetir ítems), para identificar inconsistencias en el cálculo de totales, actualización del carrito y continuidad del flujo de pago.

**Área principal explorada:** 
Carrito de compras y proceso de checkout

## Charter 2
**Título:** Exploración del comportamiento del sistema ante errores y mensajes

**Misión:** 
Explorar cómo el sistema maneja mensajes de error y validaciones durante la navegación y compra, incluyendo acciones inválidas o datos inconsistentes, para descubrir problemas de validación, mensajes engañosos y posibles inconsistencias en el flujo.

**Área principal explorada:** 
Validaciones, mensajes del sistema y manejo de errores
